AiroseUploader

Login with TikTok

AiroseUploader is a Windows desktop application. Authentication is performed using TikTok's official OAuth authorization flow.

When the user clicks "Login with TikTok" inside the desktop application, the following steps occur:

Step 1
The desktop application opens the user's default web browser.
Step 2
The browser redirects the user to TikTok's official OAuth authorization page.
Step 3
The user signs in directly with TikTok and grants permission. Credentials are entered only on TikTok's official website.
Step 4
TikTok redirects the browser to the registered callback URL.
Step 5
The desktop application receives the authorization result and displays the connected TikTok account.
Step 6
The user manually selects a video and publishes it using the official TikTok Content Posting API.
