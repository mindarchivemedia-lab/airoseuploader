OAuth Callback Endpoint

This page represents the registered OAuth callback URL used by AiroseUploader during the TikTok authentication process.

Desktop Application Notice

AiroseUploader is a Windows desktop application. After the user authorizes access on TikTok's official OAuth page, TikTok redirects the browser to this callback URL. The desktop application then completes the authentication process and securely associates the authorization with the user's local session.

This page is provided to demonstrate the registered callback endpoint used during TikTok App Review.

OAuth Flow

  1. User clicks Login with TikTok inside AiroseUploader.
  2. The desktop application opens the user's default browser.
  3. The user authenticates on TikTok's official OAuth page.
  4. TikTok redirects to this registered callback URL.
  5. The desktop application receives the authorization result.
  6. The user may now upload videos using the official TikTok Content Posting API.
