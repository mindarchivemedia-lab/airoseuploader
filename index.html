Official Website for the AiroseUploader Desktop Application

AiroseUploader is a Windows desktop application that enables creators to securely authenticate with TikTok using the official TikTok Login Kit and publish videos using the TikTok Content Posting API. The application runs locally on the user's computer and uploads media directly from the local file system. No intermediate upload servers are used.

Your videos remain on your computer until you explicitly initiate an upload. AiroseUploader never stores, proxies, or permanently hosts user media.

Official OAuth Authentication

Authentication is performed exclusively through TikTok's official OAuth authorization flow. The desktop application opens the user's default browser, allowing credentials to be entered only on TikTok's official website.

How TikTok Integration Works

AiroseUploader integrates with TikTok using the official TikTok Login Kit and Content Posting API. The authentication flow and content publishing process are initiated entirely from within the Windows desktop application. This website documents the implementation for product information and TikTok App Review.

  1. The user launches the AiroseUploader desktop application.
  2. The user manually clicks Login with TikTok inside the desktop application.
  3. The desktop application opens the user's default web browser and redirects the user to TikTok's official OAuth authorization page.
  4. The user authenticates directly with TikTok and grants permission. User credentials are entered only on TikTok's official website.
  5. TikTok redirects the browser to the desktop application's registered redirect URI, allowing the desktop application to securely complete the authentication process.
  6. The desktop application displays the authenticated TikTok account.
  7. The user selects a video file stored locally on their computer, enters a caption, and manually starts the upload.
  8. The application uploads the selected video directly to TikTok using the official Content Posting API (push_by_file workflow).
  9. No automated posting occurs. Every upload requires explicit user interaction.
