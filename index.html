Official Website for the AiroseUploader Desktop Application

AiroseUploader is a Windows desktop application that enables creators to authenticate with TikTok using the official TikTok OAuth flow and publish videos using the official TikTok Content Posting API. This website provides product information, documentation, legal policies, and demonstration videos.

Desktop Content Publishing

Select videos directly from your computer, add captions once, and publish using official platform APIs.

Local-First Architecture

Your videos remain on your computer until you explicitly initiate an upload. AiroseUploader does not proxy, host, or permanently store user videos.

Official OAuth Authentication

Authentication is performed through each platform's official OAuth flow using the user's default browser.

How TikTok Integration Works

AiroseUploader is a desktop application. The website does not perform uploads; it explains and demonstrates how the desktop application integrates with TikTok.

  1. User launches the desktop application.
  2. User clicks Login with TikTok.
  3. The application opens the user's default browser for TikTok OAuth authorization.
  4. After authorization, TikTok redirects to the registered callback URL.
  5. The desktop application receives authorization and displays the connected account.
  6. User selects a local video file and caption.
  7. The application uploads the video directly using TikTok's official Content Posting API.
  8. User manually confirms publication.
Important: All uploads are initiated manually by the user. AiroseUploader does not automate posting, scrape TikTok data, manage multiple accounts, or store user credentials.
